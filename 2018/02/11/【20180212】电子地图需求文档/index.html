电子地图需求文档

Posted on 2018-02-11
Words count in article

功能架构图

电子地图为智慧用电系列产品，用于展示智慧用电运维商下管理的项目、配电箱及其状态，并能通过表格、折线图等形式直观地展示报警情况，特定配电箱的历史数据、警情数据等信息，能在界面上进行警情处理。同时能根据报警进行实时监控与更新。

需求

界面布局

  1. 电子地图以左中右布局展示；
  2. 左侧为检索区，可以通过项目名称模糊查询；查询结果以项目名称列表的形式显示；
  3. 点击项目名称，可联动中间地图区的点，打开该项目对应点位的信息窗口
  4. 中间为地图区，地图区上显示的点位与左侧检索区一一对应；
  5. 可选择城市，加载对应城市下的项目点位

  6. 右侧为信息区，包括重点关注、统计信息。

检索区

1. 搜索框
  1. 进入页面时默认只显示搜索框，搜索框默认显示“请输入项目名称”
  2. 输入项目名称查询，支持模糊查询
  3. 输入框中有内容时，“x”按钮出现；无内容时，“x”按钮隐藏
  4. 点击搜索按钮，按钮显示按下状态，下方列表显示
  5. 点击“x”按钮，输入框清空，下方列表隐藏
2. 项目列表
  1. 点击项目，联动地图区相应点位，并打开相应信息窗口，显示该项目名称、该项目下配电箱列表、该项目状态（报警、预警、正常、离线）、配电箱总数量、各个状态配电箱数量等信息

    地图区

    1. 点位加载
  2. 地图区默认显示上次选择城市下的项目点位，若无缓存，则默认显示当前电脑IP定位的城市下的项目点位；
  1. 仅加载当前视图区，且位于当前层级的点位（向外扩张一定大小经纬度，防止轻微偏移造成点位重绘）。

  2. 默认将当前城市所有显示点位放置于当前视图区，此时可获得一个地图层级数，并根据该层级显示相应的点位;若当前城市没有点位，则弹出提示框，并将地图中心设置为当前选择的城市

  3. 点位类型分为虚拟组和真实配电箱两种类型

  4. 根据虚拟组状态显示不同点位图标
  5. 根据配电箱状态显示不同点位图标（颜色与状态的关系同微信公众号一致）
2. 打开信息窗口
  1. 点击左侧检索区搜索结果的项目，地图自动移动，打开该虚拟组的信息窗口，并让信息窗口完全显示在视图中
  2. 点击地图区的图标，打开信息窗口，并让信息窗口完全显示在视图中
  3. 虚拟组窗口：
    • 以列表形式展示该虚拟组下所有的配电箱名称，每个配电箱的状态，以及显示该虚拟组名称、虚拟组状态、配电箱总数，每种状态的数量
    • 点击列表中某一配电箱名称，能定位到该配电箱，并打开该配电箱的信息窗口

  4. 配电箱窗口：

    • 以列表形式展示该配电箱下所有的通道名称，每个通道的状态，配电箱名称，以及紧急联系人、历史数据、警情处理三个链接，点击链接，跳出该配电箱的相应信息的弹框
    • 紧急联系人：跳出弹框，显示该配电箱紧急联系人姓名、联系方式、地址
    • 历史数据：跳出弹框，显示该配电箱历史数据和报警记录（同运维平台）
    • 警情处理：跳出弹框，显示该配电箱警情处理数据，可直接进行警情处理操作（同运维平台）

  5. 发生报警

    • 通过长轮询查询报警信息。
    • 地图区上相应的虚拟组或配电箱图标状态更新，同时点位跳动3s提示；
其他
  1. 支持鼠标滚轮缩放地图；
  2. 支持鼠标左键双击放大，右键双击缩小；
  3. 定时（每5分钟）更新配电箱和虚拟组状态；

信息区

信息区分为2部分，重点关注、统计信息；

重点关注：
  1. 显示报警通道数量，包括剩余电流报警通道数量、剩余电流预警通道数量、温度报警通道数量、温度预警通道数量、烟感报警通道数量
  2. 显示离线通道数量，包括总线通讯异常配电箱数量、网络通讯异常配电箱数量；
  3. 每一项数量对应一个查询按钮，点击查询按钮，跳出弹框，显示当前选项详情
  4. 报警通道详情，显示配电箱名称、通道名称、报警类型（报警/预警）、报警时间，定位按钮。
  5. 离线通道详情，显示配电箱名称，离线通道，定位按钮。
  6. 点击定位按钮，可定位至地图的对应点位，并打开该配电箱的信息窗口
    统计信息
    显示当天的分时统计和最近7天的报警记录数量；
